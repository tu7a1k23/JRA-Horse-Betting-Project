MỘT SỐ LƯU Ý TRONG THƯ MỤC DỰ ÁN JRA HORSE BETTING
- Yêu cầu dự án: “We will build a predictive model that predicts the three fastest horses (in any order) in a race. Your customer requires criteria for all races from 2018 to now with the true positive horses at least 49.3% (i.e. if you have 600 fastest horses in 200 races, you must predict 294 horses correctly)”.
- FullProject.ipynb: Notebook phần thực hành, dùng để xử lý data, xây dựng model( training and testing), xây dựng RESTful API để dự đoán/ phân loại data mới
- Report.docx: Report EDA và quá trình xây dựng model đáp ứng yêu cầu dự án. File report này và notebook là phần chính của đồ án, có thể tham khảo qua lại giữa 2 file này.
- Data & Data Description: For data understanding and model training.
- UI cho RESTful API: Thư mục templates( thư mục này sẽ được trỏ vào khi sử dụng flask api để dự đoán kết quả)
- EDA.ipynb: code hỗ trợ report về visualizing data.
- clf_sm_nnf.pkl: Model đã tuning hyperparameter tuning hyperparameter.
- new_race_data.zip: file dùng để upload và test API.
